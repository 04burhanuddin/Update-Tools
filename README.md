# Pengguna Windows
## GIT

- Update Git
  - `git update`
  
- Remove Commit
  - `rm -rfv .git`

- Credential Manager
  - `git config --global user.name "User Name"`
  - `git config --global user.email com.example@gmail.com`
    

## Update Node.js (NPM or NODE)
- `npm install -g update-node`
- `npm install -g npm@6.9.0`
